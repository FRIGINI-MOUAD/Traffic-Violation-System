# Traffic Violation Management System

## 📌 Project Overview
This is a desktop application developed using C++ (Qt) and SQL Server for academic purposes.

The system manages traffic violations using a role-based system for police officers and citizens.

The database is local and runs on SQL Server installed on my machine.

---

## 👥 User Roles & Features

### 🚔 Police Features
- Login / Logout
- Add violation
- Update violation
- Delete violation
- Approve reports
- Reject reports

### 👤 Citizen Features
- Sign up
- Login / Logout
- Reset password
- Create report
- Update report
- Delete report
- View violations
- Search violations
- Pay fine
- View report status

---

## 🛠 Technologies Used
- C++
- Qt Framework
- SQL Server
- Qt SQL Module

---

## 🚀 Installation
1. Clone the project
2. Open in Qt Creator
3. Connect to local SQL Server
4. Build and Run
